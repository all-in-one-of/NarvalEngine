<h1>Steps to compile:</h2> </br>
<ol>
<li>git submodule init</li>
<li>git submodule update </li>
<li>mkdir build</li>
<li>cd build</li>
<li>cmake ..</li>
<li>change "glad/glad.h" to "glad/gl.h"</li>
</ol>
